# ☕ Java Basics — Learning Repository

> A clean and structured collection of Java programs covering fundamental concepts — built while learning and strengthening core Java skills.

---

## ✨ About This Repository

This repository contains my hands-on practice and learning journey through **Java basics**. It includes simple, well-structured programs that focus on understanding core programming concepts rather than just writing code.

Think of this as a **foundation layer** — everything here is meant to build strong fundamentals for advanced Java development.

---

## 🧠 Topics Covered

* 🟢 Variables & Data Types
* 🔁 Control Statements (if-else, switch)
* 🔄 Loops (for, while, do-while)
* 🧮 Operators
* 📦 Arrays
* 🔧 Methods & Functions
* 🧱 Object-Oriented Basics
* 📚 Basic Problem Solving Programs

---

java-basics/

## 🚀 Getting Started

### Prerequisites

* Java JDK (8 or above)
* Any IDE (IntelliJ, VS Code, Eclipse) or terminal

### Run a Program

```bash
javac FileName.java
java FileName
```
